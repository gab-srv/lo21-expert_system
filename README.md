Un système expert est composé d’une base de connaissances (une expertise sur un domaine donné),
une base de faits (des faits certains concernant un domaine donné) et un moteur d’inférence (un
système permettant de déduire de nouveaux faits certains à partir de la base de connaissances et de
la base de faits).
Dans ce projet, nous proposons de réaliser un mini système expert, dans lequel, les connaissances
sont des propositions logiques pouvant avoir la valeur « vrai » ou « faux ».
